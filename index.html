Uma breve introdução ao HTML

Se você está dando os primeiros passos no vasto universo da programação web, o HTML é o ponto de partida essencial. HTML, ou Hypertext Markup Language, é a linguagem fundamental para a criação de páginas web, permitindo a estruturação e formatação do conteúdo. Neste guia introdutório, exploraremos as tags principais do HTML que formam a espinha dorsal de qualquer página web.

1 - Estrutura Básica do HTML:

<!DOCTYPE html>
<html>

<head>
<title>Título da Página</title>
</head>

<body>
<h1>Este é um Título Principal</h1>
<p>Este é um parágrafo simples de texto.</p>

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

<ol>
<li>Item A</li>
<li>Item B</li>
</ol>

<a href="https://www.seusite.com">Visitar Meu Site</a>

<img src="caminho-da-imagem.jpg" alt="Descrição da Imagem">

<strong>Texto em Negrito = Texto </strong>
<i>Texto Em Itálico = Texto </i>
<mark>Texto destacado = Texto </mark>
</body>
</html>

2 - Títulos e paragrafos:

Este é um título principal

Representado pela tag: <h1> texto </h1>

Este é um subtitulo

Representado pela tag: <h2> texto </h2>

Este é um título terciário

Representado pela tag: <h3> texto </h3>

Este é um título quarternário

Representado pela tag: <h4> texto </h4>
Este é um título quinário
Representado pela tag: <h5> texto </h5>
Este é um título sênior
Representado pela tag: <h5> texto </h5>

3 - Listas ordenadas e não ordenadas:

As tags <ul> e <ol> são usadas para criar listas não ordenadas e ordenadas, respectivamente. Cada item da lista é representado pela tag <li> </li>.

    Lista ordenada
  1. Abacaxi
  2. Pera
  3. Macadamia

    Lista não ordenada
  • Melancia
  • Maçã
  • Cupuaçu

4 - Links:

5 - Imagens:

A tag <img> incorpora imagens na página. O atributo src='...' especifica o caminho da imagem, enquanto alt='descricao' fornece um texto alternativo.
